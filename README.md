# Pattern-Viewer
FPGA에서 사용되는 pat파일 컴파일러에 대한 패턴 뷰어프로그램입니다. 


Test System에서는 ALPG(algorithmic pattern generator)에서 Test Pattern을 발생시킨다.
Test Pattern 발생을 제어하는 Program을 Micro Pattern Program이라 한다.
